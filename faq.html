FAQ

Frequently Asked Questions

These answers explain how the Poorna Tech prompt library works and how viewers can use the shared resources.

What is Poorna Tech?

Poorna Tech is a short-form tech content channel that shares AI prompt ideas, creative video experiments, editing concepts, and useful digital resources.

What is this website for?

The website collects prompt documents and resources connected to Poorna Tech videos. It helps viewers find the prompt after watching a short video and use it as a learning reference.

Are the prompts free to view?

The prompt links shared on this site are intended to be publicly viewable. Some links open Google Docs or official external resources. If a document permission is incorrect, visitors can contact Poorna Tech through the social links.

Can I modify the prompts?

Yes. The prompts are meant to be adapted. Replace names, scenes, places, styles, or actions to create something original for your own use case.

Why do some prompts open in Google Docs?

Google Docs makes longer prompts easier to read, update, and share. It also allows Poorna Tech to add notes or corrections without changing the website layout every time.

Do I need a specific AI tool?

Some prompts work better with certain tools, but the structure can often be adapted. Read the prompt, understand the important details, and adjust it based on the tool you are using.
