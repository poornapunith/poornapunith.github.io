How to Write Better AI Prompts

A useful prompt gives the tool a clear creative job. Start with the subject, then add setting, action, visual style, camera or composition, and constraints.

Start with the subject

Name what should appear first. A person, object, place, or character should be clear before style words are added.

Add setting and mood

Describe where the scene happens and how it should feel. A quiet room, a bright stadium, and a rainy street produce very different outputs.

Use camera and lighting carefully

For images, mention composition and light. For video, mention motion, shot type, and pacing.

Keep one goal per prompt

Prompts become weaker when they ask for too many unrelated results. Test one idea, then revise.
